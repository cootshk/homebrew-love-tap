# Cootshk Love-tap

## How do I install these formulae?

`brew install cootshk/love-tap/<formula>`

Or `brew tap cootshk/love-tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "cootshk/love-tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
